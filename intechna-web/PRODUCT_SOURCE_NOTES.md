# Nguồn tham khảo sản phẩm cho intechna.vn

## Nguồn Ông chủ cung cấp
- CNTTShop: https://cnttshop.vn/
- Elite JSC shop: https://elite-jsc.com/shop/
- SSS Vietnam: https://sss.net.vn/

## Cách dùng an toàn
- Dùng để tham khảo danh mục, model, nhóm thiết bị và cách phân loại sản phẩm.
- Không copy nguyên văn mô tả, ảnh, bố cục chi tiết hoặc nội dung độc quyền.
- Nội dung trên intechna.vn sẽ viết lại theo hướng tư vấn/giải pháp của Intech, tránh trùng lặp SEO.

## Danh mục sản phẩm nên đưa lên web
1. Switch
   - Cisco CBS/Catalyst/Nexus
   - HPE/Aruba switch
   - Ruijie / H3C / Edgecore nếu phù hợp
   - Switch PoE, switch Layer 2/Layer 3, switch core/access

2. WiFi Access Point
   - Aruba AP series
   - Cisco/Meraki AP
   - Ruijie AP
   - UniFi AP
   - Ruckus/Cambium nếu cần mở rộng

3. Router
   - Cisco ISR
   - MikroTik RB/CCR
   - Router cân bằng tải/VPN cho doanh nghiệp

4. Firewall
   - Fortinet FortiGate
   - Cisco ASA/Firepower/Meraki MX
   - Juniper SRX
   - SonicWall/Sophos nếu cần

5. Module quang & phụ kiện mạng
   - SFP/SFP+/QSFP
   - DAC cable
   - Patch cord, patch panel, tủ rack, UPS

6. NAS / lưu trữ
   - Synology
   - QNAP
   - Thiết bị backup dữ liệu doanh nghiệp

7. Thiết bị hội nghị/phòng họp
   - Barco ClickShare C/CX series từ Elite JSC
   - Có thể đưa thành nhóm phụ: giải pháp phòng họp thông minh

## Hướng triển khai lên web
- Giai đoạn 1: chưa làm e-commerce đầy đủ, chỉ làm catalog “Liên hệ báo giá”.
- Mỗi sản phẩm có: tên nhóm, thương hiệu, ứng dụng, model tham khảo, CTA tư vấn/báo giá.
- Ưu tiên viết theo nhu cầu: văn phòng, nhà xưởng, bệnh viện/phòng khám, khách sạn, trường học.

## Trang/dữ liệu nên bổ sung
- Trang Sản phẩm hiện tại cần tách thành các nhóm: Switch, WiFi, Router, Firewall, Module quang, NAS, Phụ kiện.
